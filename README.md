# Sistema de Gerenciamento de Biblioteca

Este projeto foi desenvolvido como **Trabalho de Conclusão de Curso (TCC)** no curso técnico em Desenvolvimento de Sistemas do **Colégio Pedro II**. O objetivo foi criar uma solução para a automação de processos de empréstimos e gestão de acervo bibliotecário.

## Tecnologias Utilizadas

* **Linguagem:** C#.
* **Framework:** ASP.NET MVC (.NET Framework).
* **Banco de Dados:** SQL Server (Modelagem relacional e persistência).
* **Frontend:** HTML5, CSS3 e JavaScript.

## Funcionalidades e Destaques Técnicos

* **Arquitetura MVC:** Separação clara de responsabilidades entre Model, View e Controller para garantir organização e facilidade de manutenção.
* **Gestão de Acervo e Empréstimos:** Implementação de lógica para controle de datas, usuários e disponibilidade de livros.
* **Persistência de Dados:** Aplicação de conceitos de modelagem de dados para garantir a integridade das informações no banco de dados.

## Estrutura do Repositório

O código está organizado seguindo o padrão de projeto do ecossistema Microsoft:
* `/SOLUCAO_TCC/Controllers`: Lógica de controle e rotas.
* `/SOLUCAO_TCC/Models`: Definição das classes e regras de negócio.
* `/SOLUCAO_TCC/Views`: Interfaces de usuário.
* `BANCODEDADOSTCC.sql`: Script completo para criação e população do banco de dados SQL Server.
